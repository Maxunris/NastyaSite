## Why

Нужно реализовать адаптивный промо-сайт фестиваля «Дог порт фест» по существующему Figma-макету, чтобы команда могла публиковать событие в production без потери визуальной идентичности. Сейчас в репозитории отсутствует сайт и отсутствуют артефакты для проверки соответствия макету.

## What Changes

- Создать одностраничный адаптивный сайт (mobile-first) по макету «Дог порт фест».
- Реализовать семантическую HTML-структуру, BEM-классы и модульный CSS без UI-библиотек.
- Добавить интерактивные состояния элементов (hover/focus/active/disabled) и мягкие переходы, совпадающие по характеру с макетом.
- Подготовить структуру проекта, инструкцию запуска и чеклист визуальной сверки.
- Зафиксировать ограничения доступа к Figma-ассетам и перечень недостающих исходников для достижения строгого 1-в-1.

## Capabilities

### New Capabilities
- `dog-port-fest-landing`: Адаптивный лендинг фестиваля с секционной структурой, карточками, FAQ и блоком CTA в стилистике макета.

### Modified Capabilities
- None.

## Impact

- Affected code: новый фронтенд в корне проекта (`index.html`, `styles.css`, `script.js`, `assets/*`).
- APIs: отсутствуют.
- Dependencies: отсутствуют (vanilla stack).
- Systems: статический хостинг / любой HTTP static server.
